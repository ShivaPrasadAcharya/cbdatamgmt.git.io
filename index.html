Data Management System

How to Use This System

📁 File Format

Data files should be comma-separated .txt files with these columns:

  • caseno: Case number identifier
  • dno: Document number
  • nkp: NKP reference
  • vol: Volume number
  • date: Date (YYYY-MM-DD format)
  • cassename: Case name
  • party1/party2: Involved parties
  • bench: Bench assignment
  • remark1-5: Additional notes

🔍 Features

  • Global Search: Type in the search bar to filter across all columns
  • MySQL Filter: Advanced filtering with SQL-like conditions
  • Export: Download filtered or full data as CSV
  • Tabs: Switch between different data files

🎯 Advanced MySQL Filter

  • Add multiple conditions with AND/OR operators
  • Use operators: equals, not equals, contains, starts with, ends with, >, <, >=, <=
  • Reference columns by letter (A-N) or name
  • Write direct SQL: ColumnA = '001' AND ColumnD > 3
  • Combine conditions for complex queries

📊 Column Reference

Each column has a letter designation:

A=Case No, B=D.No, C=NKP, D=Vol, E=Date
F=Case Name, G=Party 1, H=Party 2, I=Bench
J-N=Remark 1-5
